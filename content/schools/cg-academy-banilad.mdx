<h2 id="overview">学校简介</h2>

CG Academy Banilad 校区建立于2018年，是 CG Academy 旗下的第二个校区，与 Sparta 校区同属一套师资和教学体系，两校区之间支持互转学习。

Banilad 校区位于宿务市中心，紧邻最繁华的 IT-Park 商圈，步行5分钟即可找到异国餐馆、水疗、按摩店、超市、银行等生活配套，地理位置在宿务所有英语学校中属于一流。

校园为宿务市区少见的花园式设计，配备完善的教学和生活设施。与 Sparta 校区不同，Banilad 采用**半斯巴达（Semi-Sparta）+ 自律型**管理模式，每日学习时间约8小时，课程强度较低，学员在课余有更多自由支配时间，兼顾学习与娱乐文化体验。

校区同时开设**亲子课程**和**青少年课程**，支持7岁以上儿童在家长陪同下入学，是寒暑假亲子游学的热门选择。

**基本信息：**

| 项目 | 内容 |
|---|---|
| 学校类型 | 半斯巴达式，每日晨考（单词测验） |
| 最大容量 | 约160名学生 |
| 在职教师 | 约110人 |
| 最低入学年龄 | 15岁（7~15岁需家长陪同） |
| 宵禁规定 | 周一至周四、周日：晚10点；周五、周六：晚12点 |

**适合人群：**
- 希望在市区学习、课余能自由出行的学员
- 不喜欢全天严格管制、希望保留一定自主空间的学员
- 亲子游学（家长 + 7岁以上儿童）
- 青少年暑假英语强化
- 有商务英语需求的职场人士

<h2 id="courses">课程介绍</h2>

CG Banilad 延续了 Sparta 校区的师资体系和教学质量，提供强度适中的多元课程，每堂团体课不超过4名学生，按水平分班授课。

### Semi-Sparta Course（半斯巴达课程）

Banilad 最受欢迎的课程，比全斯巴达强度更低，通过每日单词测验巩固基础，快速提升英语水平，同时保留课余自由时间。

| 课程内容 | 说明 |
|---|---|
| 4节 1:1 课程 | 阅读、听力、口语、写作 |
| 4节 1:4 课程 | CNN、商业英文、托业听力、雅思等 |
| 1节必修课 | 写作 |
| 1节选修课 | 语法、托业听力、雅思听力 |

**特殊限制：** 每天早上单词测验，若不及格则限制当天外出。

### General ESL Course（通用英语课程）

适合初学者及有意提高日常英语交流和写作能力的学员，强度最低，适合轻松过渡。

| 课程内容 | 说明 |
|---|---|
| 4节 1:1 课程 | 听力、口语、阅读、写作 |
| 2节 1:4 课程 | 商业英语、CNN等 |
| 2节选修课 | 写作、语法、托业听力、雅思听力 |

### TOEIC Course（托业课程）

适合英语基础较弱、有托业分数提升需求的学员及职场人士，每两周进行一次托业模拟考。

| 课程内容 | 说明 |
|---|---|
| 4节 1:1 课程 | 3节 TOEIC + 1节 ESL |
| 4节 1:4 课程 | 2节 TOEIC + 2节 ESL |
| 1节必修课 | 写作 |
| 1节选修课 | 语法、托业听力、雅思听力 |

**特殊限制：** 每天早上单词测验，若不及格则限制当天外出。

### Family Course（亲子课程）

适合小学、初中及高中学生在家长陪同下就读，增进亲子交流，同步提升英语基础。家长与孩子分班上课，课余一起活动。

**家长课程：**
- 4节 1:1 课程（听力、口语、阅读、写作）

**儿童课程：**
- 4节 1:1 课程（听力、阅读、语法、口语）
- 2节 1:4 课程（美式发音、商用英文、生活用语等）

### Business English Course（商业英文课程）

适合面向国际客户的商务人士，以及有商务面试、国际对话需求的学员，重点学习职场场景英语与商务礼仪。

| 课程内容 | 说明 |
|---|---|
| 4节 1:1 课程 | 商业英语 + ESL |
| 4节 1:4 课程 | 全球商业英语 + ESL |
| 1节必修课 | 写作 |
| 1节选修课 | 语法、托业听力、雅思听力 |
| 1节单词测验 | 每日必修 |

<h2 id="faq">常见问题</h2>

**Q：Banilad 和 Sparta 校区有什么核心区别？**

核心区别在于强度和位置：Sparta 每日12小时、平日禁止外出，位于郊区；Banilad 每日约8小时、有限制但宵禁相对宽松，位于市中心。两者共享师资和课程体系，可以互转。选择标准：如果需要外力约束、追求性价比 → **Sparta**；如果希望自由度高、位置便利 → **Banilad**。

**Q：亲子课程最小年龄是几岁？**

7岁以上儿童可报名亲子课程，需家长全程陪同入学。15岁以上学员可独立入读常规课程。具体开课时间有限制，建议提前3个月以上咨询。

**Q：半斯巴达课程单词测验不及格会怎样？**

当天限制外出。这是 Semi-Sparta 和 TOEIC 课程的标准规定，通过这种方式在保留学员自由度的同时维持基本学习约束。

**Q：Banilad 校区适合纯零基础学员吗？**

可以。General ESL Course 专为初学者设计，强度最低，适合零基础起步。亲子课程中的儿童班也为初学者量身安排。

**Q：千岛游学可以协助报名 CG Banilad 吗？**

可以。千岛游学提供选课、住宿安排、接机、签证手续等全程中文顾问支持，亲子游学有专属规划服务，欢迎联系咨询。
